# Notat.sh — Full Keybinding Reference (Updated)

## Legend / Conventions

### Verbs
| Key | Action |
|-----|--------|
| `s` | search (text across notes) |
| `r` | review (looped search) |
| `o` | open (predetermined file) |
| `n` | new (create file/note) |
| `p` | pick (select file/note) |

### Subjects / Note Types
| Key | Type | Notes |
|-----|------|------|
| `d` | Daily | Daily notes inbox |
| `t` | Thoughts | Quick ideas, sparks, miscellaneous thoughts |
| `a` | Actions | Action items |
| `j` | Journals | Long-running logs, diaries, work logs |
| `p` | People | Contacts, CRM-lite, interaction logs |
| `r` | Resources | Static reference material |
| `A` | Archive | Capitalized to indicate archived notes |

### Multi-letter Subjects / Variants
| Key | Meaning | Notes |
|-----|---------|------|
| `aa` | All Actions | e.g., `saa`, `raa` |
| `an` | Action Notes | e.g., `san`, `ran` |
| `ab` | Action Backlog | e.g., `sab` |

### Patterns
1. `[VERB][SUBJECT]` — most common two-letter combos.  
2. `[VERB][SUBJECT][SUBJECT_EXT]` — three-letter combos (`saa`, `ran`, etc.).  
3. Archive modifier: append capital `A` → archived notes.
